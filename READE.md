# 작업 현황 정리 (2026-04-17)

## 완료된 작업
- Mediplat 기관/기관관리자 등록 구조를 Counselman 종속에서 분리하고, Mediplat 중심 등록/로그인 흐름으로 정리.
- 기관 관리자 로그인 후 기관 사용자 추가 및 사용자 권한 관리 기능 확장.
- 병실현황판을 독립 서비스로 사용할 수 있도록 접근/권한/연동 흐름 보강.
- 세미나실 예약 서비스 기본 기능 구현.
- 세미나/세미나실 등록 및 조회/수정 흐름 보강.
- 캘린더 기반 예약 UI/UX 개선 및 반응형 대응.
- 리스트/카드 영역 비동기 갱신 개선 및 줄바꿈/가독성 UI 보정.
- favicon 교체 및 관련 정적 리소스 반영.
- 상담 화면 음성 녹음 파일 업로드에 드래그 앤 드롭 업로드(`audioDropZone`) 기능 추가.
- 음성 녹음 영역에서 기존 `[파일 업로드]` 버튼 제거 후 드롭존을 상단 배치하도록 UI 재구성.

## 작업중인 사항
- 세미나실 예약 기능의 세부 UX 다듬기(상세 모달 흐름, 상태 표시, 관리자 작업 동선).
- Counselman 일부 화면의 권한별 진입/노출 동작 점검 및 예외 케이스 정리.
- 기관/사용자/서비스 간 연동 시나리오 통합 검증.

## 작업 해야할 사항
- 권한 매트릭스(코어 관리자/기관 관리자/기관 사용자) 기반 회귀 테스트 정리.
- 세미나실 예약 알림 처리(신청/승인/반려)의 운영 정책 확정 및 고도화.
- 모바일/데스크톱 주요 화면의 E2E 점검 및 UI 폴리싱.
- 배포 전 체크리스트(로그인, 권한, 예약, 병실현황판, 음성업로드) 최종 검증.

## 업무일지 메모
- 오늘 작업은 기능 분리(서비스 단위 독립), 권한 흐름 보강, 예약/캘린더 UX 개선, 음성 업로드 UX 개선(드래그 앤 드롭)까지 진행.
- 다음 단계는 권한/시나리오 회귀 검증과 세미나실 예약 운영 완성도를 높이는 작업이 우선.
