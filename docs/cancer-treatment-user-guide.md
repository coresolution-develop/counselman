# 암센터 치료스케줄 관리 — 사용자 매뉴얼

치료실 근무자가 매일 사용하는 핵심 기능 위주로 정리했습니다.

---

## 1. 시작하기

### 로그인

별도 로그인 페이지가 없습니다. MediPlat 포털에 로그인한 뒤 "암센터 치료스케줄" 메뉴를 클릭하면 자동으로 인증되어 진입합니다.

> 세션이 만료되면 자동으로 MediPlat 로그인 화면으로 돌아갑니다. 다시 로그인 후 메뉴를 누르면 됩니다.

### 화면 구성

- **상단 메뉴바**: 스케줄 · 캘린더 · 환자 · 치료실 · 통계 · 설정
- **좌측 사이드바**: 일정 바로가기, 진행 중 건수, 최근 환자 목록
- **본문**: 선택한 페이지의 내용
- **우측 패널** (스케줄 페이지): 미니 캘린더, 다음 예약 큐, 최근 메모

오른쪽 위 `MediPlat` 버튼으로 포털로 돌아가고, `로그아웃` 버튼으로 세션을 종료합니다.

## 2. 오늘 일정 보기 (스케줄 페이지)

`/cancer-treatment-schedule`

페이지를 열면 **오늘 날짜의 시간표**가 자동으로 표시됩니다.

### 뷰 전환

상단 우측 `일 / 주 / 월` 버튼으로 표시 단위를 바꿉니다.

| 뷰 | 내용 |
|---|---|
| **일** | 시간별 환자 블록 (가장 많이 사용) |
| **주** | 월~일 7일 가로 스트립, 날짜별 건수 표시 |
| **월** | 한 달 캘린더, 날짜 셀에 건수 배지 |

주·월 뷰에서 날짜를 클릭하면 일 뷰로 전환되며 해당 날짜 시간표가 표시됩니다.

### 다른 날짜로 이동

- 우측 미니 캘린더에서 날짜 클릭
- 또는 상단 필터바의 날짜 입력란 사용
- "오늘" 버튼은 미니 캘린더 헤더에 있습니다

### 통계 카드

본문 상단 4개 카드: **오늘 전체 / 예약 / 치료완료 / 예약취소**. 필터를 적용해도 카드 숫자는 해당 날짜 전체 기준입니다.

### 필터

- **날짜**: 다른 날짜 조회
- **환자명 또는 병동**: 키워드 검색
- **치료종류 / 상태**: 드롭다운 선택
- 우측 `조회` 버튼 또는 Enter

---

## 3. 스케줄 등록

상단 우측 `+ 스케줄 등록` 버튼.

| 항목 | 설명 |
|---|---|
| **날짜 *** | 달력 클릭으로 선택. `오늘` 버튼 사용 가능 |
| **시작시간 *** | 시·분 스크롤 또는 직접 클릭 |
| **환자명 *** | 입력 시 검색 결과가 자동으로 드롭다운에 표시 |
| 병동 | 환자 선택 시 자동 채워짐. 수동 변경 가능 |
| **치료종류 *** | 입력하면 등록된 치료명이 추천됨. 새 이름도 직접 입력 가능 |
| 옵션 | 설정에서 등록한 옵션 중 선택 |
| 상태 | 기본 `예약` |
| 치료정보 | 처방 내용 메모 (인쇄물에 출력됨) |
| 메모 | 참고사항 |

`* `이 붙은 항목은 필수입니다.

### 환자명 자동완성

- 이름·차트번호·병실 일부만 입력해도 검색됩니다
- ↑/↓ 화살표로 이동, Enter로 선택, Esc로 닫기
- 일치하는 환자가 없으면 안내 메시지와 함께 "환자 관리에서 등록" 링크가 표시됩니다

### 치료종류 자동완성

- 설정 페이지에 등록된 치료명이 추천 목록으로 표시됩니다
- 목록에 없는 새 치료명은 직접 입력해서 그대로 저장할 수 있습니다

## 4. 일정 수정 · 삭제

시간표의 환자 블록을 **클릭**하면 동일한 모달이 "스케줄 수정" 모드로 열립니다.

- 필드 수정 후 `저장`
- 좌하단 `삭제` 버튼으로 일정 삭제 (확인 필요)

### 시간 변경 — 드래그&드롭

마우스로 환자 블록을 잡아 다른 시간 슬롯으로 끌어다 놓으면 시작 시간이 변경됩니다.

- 드래그 가능한 시간 슬롯은 보라색 테두리로 표시됨
- 같은 시간 슬롯에 놓으면 변경되지 않음
- 새로운 시간대(아직 예약이 없는 시간)는 모달의 시간 피커를 사용

### 상태 변경

환자 블록을 클릭해서 모달을 열고 `상태` 드롭다운에서 변경합니다.

상태 종류:
- **예약** — 진행 대기
- **치료완료** — 시술 완료
- **예약취소** — 취소됨 (블록은 흐리게 표시)

---

## 5. 일정 인쇄

상단 우측 `인쇄` 버튼.

- 현재 표시 중인 일정이 A4 한 페이지에 깔끔하게 출력됩니다
- 상단 메뉴·사이드바·필터는 자동 숨김
- 화면에는 보이지 않는 **처방내용**과 **메모**도 인쇄물에는 포함됩니다
- 취소된 예약은 환자명 옆에 `(취소)` 표기

> 종이 크기가 다르면 브라우저 인쇄 대화상자에서 배율을 조정하세요.

---

## 6. 환자 관리

`/patients`

### 환자 조회

- 검색창에 이름·차트번호·병실 키워드 입력
- 병동 필터 사용 가능

### 환자 등록

`+ 환자 등록` 버튼으로 신규 등록.

- 필수: 환자명
- 선택: 차트번호, 병실, 병동, 입원·퇴원일, 치료정보, 메모, 처방 패키지, 본인부담율, 할인 등

### 인라인 수정

목록의 셀을 더블 클릭하면 즉시 편집 모드로 바뀝니다. Enter로 저장, Esc로 취소.

## 7. 치료실 / 설정

### 치료실 관리 `/treatment-rooms`

치료실 이름과 정렬 순서를 등록·수정·삭제합니다.

### 설정 `/settings`

다음 마스터 데이터를 카테고리별로 관리합니다.

| 카테고리 | 용도 |
|---|---|
| 치료종류 | 스케줄 등록 시 추천되는 치료명 |
| 치료옵션 | 옵션 코드 + 표시 색상 |
| 치료상태 | 상태값 (기본: 예약/치료완료/예약취소) |
| 시간 슬롯 | 자주 쓰는 시간 정의 (HH:mm) |
| 병동 | 병동 코드와 이름 |
| 패키지 카테고리 | 처방 패키지 분류 |

> 자주 등록되는 항목을 미리 설정해 두면 스케줄 입력이 빨라집니다.

## 9. 실시간 동기화

스케줄 페이지 우측 상단에 **실시간 연결** 표시가 있습니다.

- 🟢 **실시간 연결** — 다른 사용자가 일정을 추가·수정하면 즉시 화면에 반영
- 🟡 **자동 조회 전환** — 연결이 끊겨 15초 간격으로 자동 새로고침 중
- 🟡 **연결 중** — 초기 연결 시도

연결이 끊겨도 데이터 손실은 없으며, 잠시 뒤 자동으로 다시 연결됩니다.

## 10. 자주 묻는 질문

**Q. 다른 사람이 등록한 일정도 보이나요?**
같은 병원 코드의 모든 사용자가 동일한 스케줄을 공유합니다. 등록·수정 즉시 다른 사람 화면에도 반영됩니다.

**Q. 인쇄가 깨져요.**
브라우저 인쇄 대화상자에서 "여백: 보통", "배율: 100%" 또는 "기본"으로 설정하세요. 종이가 A4가 아니라면 배율을 조정합니다.

**Q. 드래그가 안 돼요.**
태블릿·휴대폰에서는 터치 드래그가 지원되지 않습니다. 데스크탑(마우스)에서 사용해 주세요. 대안으로 환자 블록을 클릭해 모달에서 시간을 변경할 수 있습니다.

**Q. 환자가 자동완성에 나오지 않아요.**
환자 관리 페이지에서 등록되어 있는지 확인하세요. 등록되지 않은 환자는 검색되지 않습니다. 모달의 안내 메시지에서 환자 관리 페이지로 이동할 수 있습니다.

**Q. 화면이 멈춘 것 같아요.**
우측 상단의 실시간 연결 표시를 확인하고, 페이지 새로고침(F5)을 시도해 보세요. 그래도 안 되면 MediPlat 포털에서 다시 로그인하세요.
